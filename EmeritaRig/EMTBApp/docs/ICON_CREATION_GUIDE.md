## Step-by-Step Creation Guide

### Phase 1: Master Icon Design (1024×1024)

#### Step 1: Set Up Document
```
1. Create new document:
   - Size: 1024×1024 pixels
   - Color mode: RGB
   - Resolution: 72 DPI
   - Background: Transparent

2. Add guides:
   - Center guidelines (512×512)
   - Safe area: 100px margin
   - Content area: 824×824 pixels
```

#### Step 2: Create Background
```
1. Draw circle: 924×924 pixels (50px margin)
2. Apply gradient:
   - Type: Radial
   - Center: 30% from top-left
   - Colors: #3b82f6 → #1e40af
   - Opacity: 100%

3. Add subtle shadow:
   - Inner glow effect
   - Color: rgba(255,255,255,0.2)
   - Size: 20px
   - Position: Top-left highlight
```

#### Step 3: Design Star of Life
```
1. Create six-pointed star:
   - Outer diameter: 600px
   - Inner diameter: 200px
   - Points: 6, equal spacing
   - Color: Pure white (#ffffff)
   - Stroke: None

2. Add Rod of Asclepius:
   - Staff: 8px width, 400px height
   - Snake: Curved path, 6px stroke
   - Position: Center of star
   - Color: White (#ffffff)
```

#### Step 4: Add Typography
```
1. Create "EMS" text:
   - Font: Helvetica Neue Bold
   - Size: 120px
   - Color: White (#ffffff)
   - Position: Bottom third of circle
   - Letter spacing: +2px

2. Add text effects:
   - Drop shadow: 2px offset, 50% opacity
   - Slight emboss for depth
```

#### Step 5: Medical Cross Accent
```
1. Create small cross:
   - Size: 80×80 pixels
   - Stroke: 16px width
   - Color: Emergency red (#dc2626)
   - Position: Top-right of star

2. Add white outline:
   - 2px white stroke around red cross
   - Ensures visibility on blue background
```

### Phase 2: Size Variations

#### Large Sizes (152px and above)
```
- Maintain all design elements
- Keep text readable
- Preserve symbol details
- Use full color palette
```

#### Medium Sizes (76px - 120px)
```
- Simplify star details slightly
- Maintain "EMS" text
- Keep medical cross accent
- Reduce shadow effects
```

#### Small Sizes (29px - 58px)
```
- Remove "EMS" text
- Focus on star symbol only
- Increase contrast
- Simplify rod details
- Bold, clear shapes only
```

### Phase 3: Platform Optimization

#### iOS Specific
```
1. Corner Radius: Applied automatically by iOS
2. Safe Area: Keep content 10% from edges
3. Background: Ensure it extends to edges
4. Testing: Use iOS simulator for validation
```

#### Android Adaptive Icons
```
1. Foreground Layer (192×192):
   - Star of Life symbol only
   - White on transparent
   - Safe zone: 66dp radius
   - No background elements

2. Background Layer (192×192):
   - Blue gradient only
   - Full bleed design
   - No critical content
   - Compatible with masks
```
